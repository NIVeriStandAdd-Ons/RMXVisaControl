3. Known Issues

    Instrument Multichannel Setup
  • The instruments must be configured for multichannel setup using the instrument front panel, selecting a master and slave units.
  • Then instruments must be restarted, turning on the slave units first and the master unit last.
  • Please see the users manual and programmers manual for details on how to configure a multichannel setup.
